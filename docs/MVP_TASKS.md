## TASK-05: Open-Meteo API 取得フック

**ステータス**: [ ]
**担当**: Generator
**依存**: TASK-03

### やること

- Open-Meteo API（`https://api.open-meteo.com/v1/forecast`）から現在地の気象データを取得する React カスタムフックを作る
- 取得するパラメータ: `precipitation`（降水量）、`cloud_cover`（雲量）の現在時刻分
- フックが返す値: 降水量・雲量・取得ステータス・エラーメッセージ
- 取得した降水量・雲量から「虹が出やすいか」を判定するロジックも含む

判定基準:
- 降水量 > 0 かつ 雲量 < 80 → 「虹が出やすい」
- 雲量 >= 80 → 「曇りのため太陽が見えにくい」
- 降水量 = 0 → 「雨が降っていません」

### 完了条件

- ブラウザで `npm run dev` を開き、開発者コンソールに取得した降水量・雲量・判定結果が出力される
- ネットワークタブで `api.open-meteo.com` へのリクエストが確認できる
- API 取得成功時・失敗時ともにコンソールで状態が確認できる
- `npm run build` が成功する

### 変更ファイル（予定）

- `src/hooks/useWeather.ts`
- `src/app/page.tsx`（確認用の console.log を追加）

### commit メッセージ（予定）

```
feat: add Open-Meteo weather data hook
```
